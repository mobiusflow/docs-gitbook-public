# Welcome to Version 3!

MobiusFlow V3 has been built from the ground up. We have listened to your feedback and add a host of new features to make MobiusFlow more powerful and easier to use.

Tagging, we have added it, custom object profiles, there too, and to make you even more productive we have added the ability for you to write your own data pre-processors and codecs. We are excite about these new features and are sure you will be too!
